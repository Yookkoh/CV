# Notes — Design & UX

## Visual Style (Apple-like)
- Minimal palette (light + dark)
- Generous whitespace
- Crisp typography (system-ui / SF Pro)
- Subtle glass panels:
  - blur 12–18px
  - hairline border
  - soft shadow

## Motion Guidelines
- No bounce; use smooth ease-out.
- Keep movements small (8–16px).
- Route transitions: fade + slight translate + mild blur.
- Respect prefers-reduced-motion.

## Mobile Dock
- Fixed bottom dock, frosted background.
- 4–5 main items.
- Active indicator is subtle (thin underline/dot).
- Tap feedback: small scale down then release.
